xcode launcher
====

Launch xcode from command line.

## Usage

Launch xcode project
```
$ xcode
$ xcode sample
$ xcode sample/sample.xcodeproj
$ xcode sample1 sample2
```

Specify xcode version

```
$ xcode 
$ xcode 5
$ xcode 6-Beta6
```

Check which xcode version is installed
```
$ xcode -l
```
